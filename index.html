MODELO DE MENSURAÇÃO

IAM Control Effectiveness Model

Ingere evidência de qualquer plataforma IAM (JSON ou CSV), calcula scores de efetividade por controle ISO, avalia risco com mapeamento LGPD e gera decisões semi-automáticas para confirmação do analista. Histórico de múltiplos ciclos com rastreabilidade de cada ingestão via hash SHA-1.

5.35.155.18 5.338.28.3 LGPD
Abrir dashboard →
  • Ingestão de evidência via JSON (cycles.json) ou CSV (metrics.csv) — agnóstico de plataforma
  • Score de efetividade por dimensão: RBAC · PAM · SoD · Review · Evidence
  • Avaliação por motor de risco ISO 27001 — thresholds por controle com mapeamento LGPD
  • Decisão sugerida (REMEDIATE / ESCALATE / MITIGATE / ACCEPT) confirmada pelo analista
  • Gráfico de evolução histórica por ciclo
  • Rastreabilidade de origem por ingestão — plataforma registrada com hash SHA-1
  • Manifesto de auditoria com hash SHA-1 por importação
  • Export de evidência: JSON snapshot · CSV métricas · JSON decisões · HTML relatório auditável

Avaliação de risco — ISO 27001 + LGPD
LGPD

Modelo de risco documentado em docs/risk-model.md O modelo avalia cada métrica contra thresholds definidos por controle ISO e mapeia o impacto regulatório LGPD correspondente. Falhas críticas em SoD (5.3), PAM (8.2), revisão (5.18) e evidência (5.33) geram achados com artigo LGPD, tipo de risco regulatório e decisão sugerida. O analista confirma ou ajusta cada decisão antes do export — garantindo rastreabilidade da decisão humana sobre o risco identificado.
